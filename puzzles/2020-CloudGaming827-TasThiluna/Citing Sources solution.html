Solution

Four 8×3 grids are presented, each accompanied by two letters with ellipsis between them. These are the first and last letters of the name of a module creator that you can use to take 4 modules from each row. This allows us to form four 4×3 grids.

The first letter of every unused module spells out APPLY THE ADJACENT LETTERS RULES OBTAIN BRAILLE LETTERS. Doing so spells out the solution.
