Keep Talking and Nobody Explodes Mod Torture

On the Subject of Torture

*sighs* “Here we go again.”

This module consists of a 4×4 grid with a chessboard-like pattern of black and azure tiles, each having a number on it.

Pressing any tile will add an offset to 5 to 7 other tiles. For example, pressing A1 can add 4 to C2, 7 to D4, 3 to D1, etc. After the offsets are added, the rightmost digit of each affected tile is taken as its value.

To disarm this module, press the tiles such that each tile displays the same number.

Note: There is a button at the same position as the status light. This button is meant for checking answers/training only. Pressing it will put a line in the logfile showing the current state’s solution, but will also turn the module red, indicating that the button has been pressed, and will also invalidate any challenge bomb solve with it pressed.
