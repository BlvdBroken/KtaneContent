On the Subject of Next In Line

Unlike elementary school, cutting in line is allowed.

  • To begin this module, cut the first wire.
  • In the table below, use the color of the previously cut wire as the column in conjunction of the current wire number to determine the color of the wire that you should cut next. Not cutting a wire will still increase the iteration count.
  • A cut will only be processed when the button is pressed, and thusly any wire that was incorrectly cut/skipped will only incur a strike when the button is pressed.
  • Incorrectly cutting/skipping a wire will reset this module back to iteration 2.
